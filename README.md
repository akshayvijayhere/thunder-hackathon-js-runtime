# Thunder Hackathon 2.0 – JavaScript Runtime

## Overview

This project is a Python-based JavaScript runtime.

It accepts JavaScript source code as input, executes it using a JavaScript execution engine, and prints the output to stdout.

## Requirements

* Python 3.x
* Node.js

## Usage

Run using a JavaScript file:

python3 runtime.py program.js

Or:

python3 runtime.py < program.js

## Example

Input:

console.log("Hello World");

Output:

Hello World

## Supported Features

* Variables (let, const)
* Arrays
* Objects
* Functions
* Arrow Functions
* Loops
* Conditionals
* Array Methods
* String Methods
* Math Object
* Date Object
* Spread and Rest Operators
